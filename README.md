☕ Coffee Shop Sales Analysis Dashboard

📌 Project Overview
This project focuses on analyzing coffee shop sales data to uncover meaningful business insights and improve overall store performance. The dashboard was built using Power BI to visualize sales trends, customer behavior, product performance, and store-wise analysis.

🎯 Objective
The main objective of this project is to analyze retail sales data and gain actionable insights that help enhance the performance of the coffee shop business.

📊 Dashboard Features
✔ Key Metrics
Total Sales
Total Footfall
Average Bill Per Person
Average Orders Per Person

✔ Analysis Performed
Sales variation by day and hour
Peak sales hours identification
Monthly sales analysis
Store location performance comparison
Product category distribution
Product size distribution
Top-selling products analysis
Weekday order analysis

📈 Insights Generated
Identified peak ordering hours during the day
Analyzed customer purchasing behavior
Compared sales and footfall across store locations
Determined best-selling coffee products
Evaluated category-wise contribution to total sales
